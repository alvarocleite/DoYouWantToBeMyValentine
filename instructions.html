How to Personalize

You can add your Valentine's name directly in the URL to make it special!

Just add ?name= followed by the name at the end of the URL.

Example:

If you are using the name Sarah:

https://alvarocleite.github.io/DoYouWantToBeMyValentine/?name=Sarah

Try it out and send it to someone special!
